## 🛒 Project 1 — Blinkit Sales Analysis Dashboard

### Overview
An interactive sales analytics dashboard analyzing **$1.20M** in 
total sales across **8,523 items** for Blinkit — India's last minute 
delivery app — tracking outlet performance, item categories, and 
customer ratings.

---

### 📊 Key KPIs

| Metric | Value |
|---|---|
| **Total Sales** | $1.20M |
| **Average Sales** | $141 |
| **Total Items** | 8,523 |
| **Average Rating** | 3.9 |

---

### 📉 Key Visualizations

| Visual | Insight |
|---|---|
| **KPI Cards** | Total sales, avg sales, items, rating |
| **Sales by Outlet Type** | Supermarket Type 1 leads at $787.55K |
| **Sales by Location Tier** | Tier 3 — $472.13K, Medium outlets — $507.90K |
| **Category Analysis** | Fat content, item type breakdown |
| **Outlet Establishment Trend** | Year-wise outlet growth |
| **Dynamic Filter Panel** | Outlet type, size, location filters |

---

### 🔑 Key Insights

| Finding | Detail |
|---|---|
| **Top Outlet Type** | Supermarket Type 1 — $787.55K |
| **Top Location** | Tier 3 locations — $472.13K |
| **Best Category** | Fruits and Vegetables, Snack Foods |
| **Medium Outlet Sales** | $507.90K |
| **Grocery Store Sales** | $151.94K |

---

### ⚙️ Power BI Features Used

| Feature | Where Used |
|---|---|
| **Power Query ETL** | Data cleaning and transformation |
| **DAX Measures** | CALCULATE, AVERAGEX, Time Intelligence |
| **KPI Cards** | Sales, quantity, rating tracking |
| **Donut Charts** | Outlet size distribution |
| **Bar Charts** | Item type and outlet comparison |
| **Slicers** | Outlet type, size, location filters |

## 🚗 Project 2 — Uber Ride Analysis Dashboard

### Overview
A comprehensive Power BI dashboard analyzing **117K Uber trips** 
tracking total fare, trip duration, distance, payment methods, 
shift patterns, and pickup time trends across multiple locations 
in New York City.

---

### 📊 Key KPIs

| Metric | Value |
|---|---|
| **Total Trips** | 117K |
| **Total Fare** | $1.8M |
| **Total Duration** | 1.9M mins |
| **Total Distance** | 393.9K miles |
| **Night Shift %** | 8.7% |

---

### 📉 Key Visualizations

| Visual | Insight |
|---|---|
| **Total Trips by Location** | New Yorkus Bowne Park highest volume |
| **Trips by Payment Mode** | Cash (67.85%), Uber Pay (32.21%), Google Pay, Amazon Pay |
| **Trips by Shift** | Day (91.3%), Night (8.7%) |
| **Total Trips by Days** | Wednesday and Thursday peak days |
| **Trips by Pickup Time** | Peak hours 12PM-6PM |
| **Top Location** | LaGuardia Airport — $14.956 avg fare |
| **Top Day** | Sunday — $14,884.5 total fare |
| **Dynamic Filter Panel** | Month, Year, Day Name, Location filters |

---

### 🔑 Key Insights

| Finding | Detail |
|---|---|
| **Dominant Payment** | Cash dominates at 67.85% of trips |
| **Peak Day** | Wednesday and Thursday highest trips |
| **Peak Hours** | Afternoon 12PM-6PM busiest period |
| **Night vs Day** | 91.3% day trips vs 8.7% night shift |
| **Top Location** | LaGuardia Airport highest avg fare |
| **Top Fare Day** | Sunday generates highest total fare |

---

### ⚙️ Power BI Features Used

| Feature | Where Used |
|---|---|
| **Power Query ETL** | Trip data cleaning and transformation |
| **DAX Measures** | Fare calculations, shift percentages |
| **KPI Cards** | Trips, fare, duration, distance, night shift |
| **Bar Charts** | Location wise trip volume |
| **Donut Charts** | Payment mode and shift distribution |
| **Line Charts** | Trips by days and pickup time trends |
| **Slicers** | Month, year, day, location filters |
| **Reset Button** | Clear all filters at once |
