### 🛒 Blinkit Sales Analysis Dashboard
**Tools Used:** Power BI, Power Query, DAX

**Overview:**
An interactive sales analytics dashboard built for Blinkit — India's last minute delivery app — analyzing $1.20M in total sales across 8,523 items, multiple outlet types, sizes, and locations.

**Key Highlights:**
- 📊 Tracked **$1.20M** total sales, **$141** average sales, and **3.9** average customer rating
- 🏪 Identified **$507.90K** in Medium outlet sales and **$472.13K** in Tier 3 location sales
- 🥇 Pinpointed top performing outlet — Supermarket Type 1 at **$787.55K**
- 🔍 Built dynamic filter panels for Outlet Type, Outlet Size, and Item Type
- 📈 Developed custom DAX measures analyzing fat content, item type, and outlet establishment trends
- 🗺️ Multi-page dashboard tracking **10+ key business KPIs**

**What I Did:**
- Cleaned and transformed raw dataset using **Power Query ETL workflow**
- Built **custom DAX measures** for sales aggregations and trend analysis
- Designed **interactive visualizations** including donut charts, bar charts, and KPI cards
- Added **dynamic slicers** for real-time filtering across all visuals
